# Skin Lab 免费7天皮肤健康体验

## 项目概述
高级感Landing Page，专注于皮肤检测体验预约，采用限量周期性名额系统。

## 技术架构
- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **后端**: Google Apps Script
- **数据库**: Google Sheets
- **部署**: GitHub Pages

## 功能特色
1. 高级视觉设计，营造专业实验室氛围
2. 周期性名额系统（每周50位，周日重置）
3. 智能表单弹出（滚动深度触发+时间备份）
4. 实时名额计数
5. 数据提交到Google Sheets
6. 提交成功页面
